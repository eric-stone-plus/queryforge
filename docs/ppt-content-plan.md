# QueryForge 路演 PPT 内容方案

> 给投资人/评委看的商业路演，不是技术汇报。
> 不提 SQL、不提技术栈、不提评分预估。

---

## 故事线

```
痛点（为什么做）
  → 方案（怎么做）
    → 产品（做成什么样）
      → 数据验证（用数据说话）
        → 商业模式（怎么赚钱）
          → 市场规模（能赚多少）
            → 团队（谁来做）
              → 行动（现在要什么）
```

核心叙事张力：**分析师被困在重复取数里 vs 业务部门永远在等数据**。
一句话：「让分析师做分析师该做的事，让业务自己拿到数据。」

## 第 2 页：痛点

**标题**：数据分析师的困境

**核心信息**（一个大数字 + 两个场景）：

大数字：
> **80%**
> 数据分析师的时间花在重复取数上

两个场景对比（左右分栏）：

| 左：分析师的一天 | 右：业务团队的一天 |
|---|---|
| 「帮我拉一下上个月各地区的 GMV」 | 周一提需求，周三还没拿到数据 |
| 「口径改一下，要按下单时间不是支付时间」 | 拿到数据发现字段不对，又要重来 |
| 「再加一个退货率的列」 | 一个简单报表，改了 5 版 |
| 每天 2-3 小时在重复拉报表 | 排期等待 1-3 天是常态 |

**视觉元素**：
- 顶部大数字 `80%`，琥珀金色，48pt Bold
- 下方两个区块，左侧珊瑚红边框（分析师视角），右侧电光蓝边框（业务视角）
- 每个场景用简洁的文字卡片，不用图标

**数据支撑**：「10,000 笔订单、8 个地区、20 个品类——真实企业级数据量下，这个问题更突出。」

## 第 3 页：解决方案

**标题**：分析师建指标，业务自助取数

**核心信息**：

一句话：
> 从「取数工具人」到「数据架构师」

流程（4 步，横向排列）：

```
分析师定义指标  →  业务用自然语言提问  →  系统自动生成图表  →  异常自动预警
```

每步一行说明：
1. **分析师**：调整底层数据、定义指标口径、配置看板（一次性工作）
2. **业务人员**：用中文描述需求，像和同事说话一样
3. **智能引擎**：理解意图，自动匹配数据源，生成可视化
4. **输出**：图表、异常提示、决策建议

**视觉元素**：
- 4 个步骤用左侧色条卡片（蓝→蓝→金→绿）
- 步骤之间用细箭头连接（`→` 字符即可，不要花哨图形）
- 底部一句话：「分析师从此不再反复拉报表」

**话术要点**：强调「角色转变」——分析师不是被取代，而是被解放去做更有价值的事。

## 第 4 页：产品演示

**标题**：从提问到看板，10 秒

**核心信息**（3 个真实场景）：

| 场景 | 用户提问 | 输出 |
|------|---------|------|
| 基础查询 | 「各地区月度销售额趋势」 | 8 地区 × 12 个月折线图 |
| 复杂分析 | 「哪个品类利润率最高？」 | 品类利润率排名柱状图 |
| 自由提问 | 「复购率最高的用户是谁？」 | 用户复购排名表 |

关键卖点：
- 不需要懂任何技术，自然语言提问
- 自动选择合适的图表类型
- 指标可以保存、复用，一键刷新

**视觉元素**：
- 中间放一张产品截图（真实界面，展示自然语言输入 + 图表输出）
- 左右两侧各一个场景卡片，简洁文字说明
- 截图加 1pt 细边框（`#2a3f5f`），不做圆角阴影
- 底部用小字标注：「真实数据：10,000 笔订单 · 500 商品 · 8 地区」

**话术要点**：这是唯一需要简短演示的页面——如果条件允许，点一下产品展示实际效果。

## 第 6 页：商业模式

**标题**：怎么赚钱

**核心信息**：

定价模型（SaaS 订阅制）：

| 版本 | 价格 | 包含 |
|------|------|------|
| 团队版 | ¥299/月/团队 | 最多 10 人，50 个指标 |
| 企业版 | ¥999/月/团队 | 无限人数，无限指标，私有部署 |

收入逻辑：
- 目标客户：有数据团队但业务侧频繁提需求的中型企业（100-1000 人）
- 获客路径：分析师试用 → 团队采用 → 企业采购
- 留存逻辑：指标一旦定义，迁移成本高，自然锁定

**视觉元素**：
- 左侧：定价表格，简洁无装饰
- 右侧：一个简单的获客漏斗示意（文字即可，不用图形）
  ```
  分析师注册试用
    ↓
  团队日常使用
    ↓
  企业级采购
  ```
- 底部一行小字：「核心壁垒：指标资产沉淀 + 口径标准化」

## python-pptx 实现要点

1. **去掉所有 `add_label` 调用**（AI 风格胶囊标签）
2. **大数字用 48-54pt Bold**，正文 14pt，层次拉大
3. **卡片统一用 `#1F3056` 背景**，无边框无阴影
4. **装饰线**：标题左侧 4px 色条（琥珀金或电光蓝）
5. **列表用 `—` 前缀**，不用圆点
6. **产品截图**：用 `slide.shapes.add_picture()` 插入 PNG
7. **配色统一**：BG=`#1B2A4A`, TEXT=`#FFFFFF`, SUBTLE=`#B0BEC5`, ACCENT=`#FFB300`, BLUE=`#2196F3`
