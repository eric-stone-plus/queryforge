# QueryForge — 3 分钟路演脚本

## 开场（30 秒）

"每个企业都有数据，但不是每个人都会写 SQL。业务团队提一个需求，数据团队排期、改口径、拉报表——通常要等 3-5 天。"

"我们做了一个 AI 数据分析智能体。用自然语言提问，自动生成 SQL，实时出图表。从提问到看板，只需要 10 秒。"

## Demo 演示（2 分钟）

### 场景 1：基础查询（30 秒）
点击"各地区月度销售额趋势"芯片
- 展示：AI 推理过程 → SQL 生成 → 折线图渲染
- 强调：240 个数据点，8 个地区，12 个月趋势
- 亮点：点击"查看推理过程"展示 AI 思考链

### 场景 2：复杂分析（30 秒）
点击"哪个品类利润率最高？"芯片
- 展示：利润率计算公式 SUM(oi.quantity*oi.unit_price*(1-oi.discount))
- 强调：AI 自动选择正确的收入公式，不用 total_amount
- 亮点：SQL 安全校验——只允许 SELECT，防止注入

### 场景 3：自由提问（30 秒）
手动输入："复购率最高的用户是谁？"
- 展示：AI 理解"复购"概念 → 生成购买次数统计
- 强调：自然语言理解能力，不需要用户懂 SQL

### 场景 4：指标保存（30 秒）
点击"保存指标"按钮 → 侧边栏出现保存的指标
- 展示：指标可以保存、复用、一键重新查询
- 强调：分析师的工作流——不是一次性查询，是持续的数据探索

## 收尾（30 秒）

"这是一个可商业化的产品。企业数据团队每天花 2-3 小时拉报表，用 QueryForge 可以缩短到分钟级。"

"技术栈：Next.js + SQLite + MiMo 大模型，单次 API 调用，2-4 秒响应。SQL 安全审计、指标持久化、图表自动选择——这不是 demo 玩具，是可以部署的 SaaS。"
